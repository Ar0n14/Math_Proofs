# Math proofs for the exam of June 2025

### CONTENTS
- `main.pdf`: Rendered output of the LaTeX document.
- `main.tex`: LaTeX source code for the document.
- `main.log`: Compilation log file.
- `main.aux`: Auxiliary file used by LaTeX.
- `main.fls`: File list generated during LaTeX compilation.
- `main.fdb_latexmk`: Database file used by `latexmk` to track dependencies and determine if recompilation is needed.
- `main.synctex.gz`: SyncTeX file allowing synchronized navigation between the PDF and source `.tex` files in compatible LaTeX editors.
- `LICENSE`: MIT license
